MARKDOWN

¿QUÉ ES?

Markdown o MD es un lenguaje de marcado ligero creado por John Gruber. Permite escribir documentos en un formato de texto plano, fácil de leer, escribir, y luego convertirlos en documentos HTML válidos.

¿CÓMO SE UTILIZA?

Markdown se utiliza en muchos lugares donde es necesario dar estilo o estructurar texto plano. Los desarrolladores y redactores técnicos a menudo usan Markdown para escribir documentación debido a su legibilidad y fácil integración con sistemas de control de versiones como Git.
